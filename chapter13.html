This chapter talks about lists and tables

You can put things in a list like this:

  • Item 1
  • Item 2
  • Item 3

You can also put things in a table like this:

Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3
Row 3, Cell 1 Row 3, Cell 2 Row 3, Cell 3
